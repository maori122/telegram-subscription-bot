# 🤖 Telegram Bot для рассылки (Cloudflare Workers)

Современный Telegram бот с интерактивными кнопками для рассылки сообщений. Работает на Cloudflare Workers - бесплатно и без ограничений!

## ✨ Особенности

- 🎯 **Интерактивные кнопки** - удобное управление через inline-клавиатуру
- 🔒 **Безопасная авторизация** - система паролей для админов
- 📢 **Массовая рассылка** - отправка сообщений всем пользователям
- ✉️ **Персональные сообщения** - отправка конкретным пользователям
- ☁️ **Cloudflare Workers** - бесплатный хостинг 24/7
- 💾 **Cloudflare KV** - надежное хранилище данных
- ⚡ **Мгновенный отклик** - edge computing

## 🚀 Быстрый старт

### 1. Установка

```bash
npm install -g wrangler
wrangler login
```

### 2. Настройка

Создайте KV namespace:
```bash
wrangler kv:namespace create "USERS"
```

Обновите `wrangler.toml` с полученным ID.

### 3. Секреты

```bash
wrangler secret put BOT_TOKEN
# Введите токен от @BotFather

wrangler secret put ADMIN_PASSWORD
# Введите: Yp~?L*W<a"v/(~mFm&[&
```

### 4. Деплой

```bash
wrangler deploy
```

### 5. Webhook

```bash
curl "https://api.telegram.org/botYOUR_TOKEN/setWebhook?url=https://YOUR_WORKER_URL"
```

## 📱 Использование

### Для пользователей:
1. Найдите бота в Telegram
2. Отправьте `/start` для регистрации
3. Используйте `/help` для справки

### Для администраторов:
1. Отправьте `/admin Yp~?L*W<a"v/(~mFm&[&`
2. Появятся кнопки управления:
   - 👥 **Список пользователей** - просмотр всех зарегистрированных
   - 📢 **Рассылка всем** - массовая отправка сообщений
   - ✉️ **Отправить конкретному** - выбор пользователя из списка

## 🎮 Интерфейс с кнопками

После авторизации админ видит удобное меню с кнопками:

```

[📢 Рассылка всем      ]
```
