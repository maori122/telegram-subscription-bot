# 🎯 НАЧНИТЕ ЗДЕСЬ

## Что делает этот бот?

Это система для работы с рекламными подписками:

1. **Пользователи регистрируются** - нажимают кнопку ✅
2. **Админ рассылает ссылки** - одна кнопка для всех
3. **Пользователи подписываются** - переходят по ссылке
4. **Отправляют скриншот** - в бота
5. **Получают подтверждение** - автоматическое сообщение
6. **Админ управляет** - может +1, -1 или обнулить подписки

## 🚀 Быстрый старт (3 шага)

### Шаг 1: Установите Wrangler

```bash
npm install -g wrangler
```

### Шаг 2: Запустите автоустановку

**Windows:**
```bash
setup.bat
```

**Linux/Mac:**
```bash
chmod +x setup.sh
./setup.sh
```

### Шаг 3: Установите webhook

После деплоя:
```bash
curl "https://api.telegram.org/botВАШ_ТОКЕН/setWebhook?url=https://ВАШ_WORKER_URL"
```

## ✅ Готово!

Теперь проверьте бота:

1. Отправьте `/start`
2. Нажмите "✅ Зарегистрироваться"
3. Авторизуйтесь как админ: `/admin Yp~?L*W<a"v/(~mFm&[&`
4. Появятся кнопки управления!

## 📱 Как использовать

### Сценарий работы:

**1. Пользователь регистрируется:**
```
Бот: "Привет! Я бот, который выдает рекламные ссылки"
     [✅ Зарегистрироваться]

Пользователь нажимает кнопку
Бот: "Отлично! Вы зарегистрированы"
```

**2. Админ рассылает ссылку:**
```
Админ: Нажимает "🔗 Разослать ссылку"
Админ: Отправляет ссылку (например: https://t.me/channel)

Все пользователи получают:
"🔗 Новая рекламная ссылка:
https://t.me/channel

После подписки отправьте скриншот в этот чат."
```

**3. Пользователь подписывается и отправляет скриншот:**
```
Пользователь: [отправляет фото]

Бот автоматически:
"Зафиксировано 1 подписок"
```

**4. Админ управляет подписками:**
```
Админ: Нажимает "👥 Список пользователей"
Админ: Выбирает пользователя
Админ: Может нажать:
  - ➕ +1 подписка
  - ➖ -1 подписка
  - 🔄 Обнулить
```

## 🎮 Админ-панель

После авторизации (`/admin пароль`) появляются кнопки:

### 👥 Список пользователей
- Показывает всех зарегистрированных
- Отображает количество подписок каждого
- Можно выбрать для управления

### 🔗 Разослать ссылку
- Отправляет ссылку всем пользователям
- Автоматически добавляет инструкцию
- Показывает статистику доставки

### 💬 Настроить ответ
- Кастомный текст после скриншота
- Используйте `{count}` для подстановки числа
- Пример: "Зафиксировано {count} подписок! 🎉"

---

## 🔧 Управление подписками

Для каждого пользователя доступно:

**➕ +1 подписка** - увеличить счетчик
- Если пользователь забыл отправить скриншот
- Если нужно добавить вручную

**➖ -1 подписка** - уменьшить счетчик
- Если была ошибка
- Если нужно скорректировать

**🔄 Обнулить** - сбросить в 0
- Начать отсчет заново
- Очистить статистику

---

## ❓ Частые вопросы

**Q: Как изменить текст приветствия?**
A: Отредактируйте `src/index.js`, строка с "Привет! Я бот..."

**Q: Можно ли добавить несколько админов?**
A: Да, каждый кто знает пароль может авторизоваться

**Q: Где хранятся данные?**
A: В Cloudflare KV (постоянное хранилище)

**Q: Что если пользователь отправит не скриншот?**
A: Бот обрабатывает только фото, текст игнорируется
