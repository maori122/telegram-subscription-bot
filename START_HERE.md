## Что изменилось?

✅ Бот переписан на **JavaScript**  
✅ Добавлены **интерактивные кнопки**  
✅ Готов для **Cloudflare Workers** (бесплатный хостинг 24/7)  
✅ Использует **Cloudflare KV** для хранения данных  

## 🚀 Быстрый старт (3 шага)

### Шаг 1: Установите Wrangler

**Windows:**
```bash
npm install -g wrangler
```

**Linux/Mac:**
```bash
npm install -g wrangler
```

### Шаг 2: Запустите автоматическую настройку

**Windows:**
```bash
setup.bat
```

**Linux/Mac:**
```bash
chmod +x setup.sh
./setup.sh
```

Скрипт автоматически:
- Войдет в Cloudflare
- Создаст KV namespace
- Настроит секреты
- Задеплоит бота

### Шаг 3: Установите webhook

После деплоя скопируйте URL вашего worker и выполните:

```bash
curl "https://api.telegram.org/botВАШ_ТОКЕН/setWebhook?url=https://ВАШ_WORKER_URL"
```

Или откройте в браузере:
```
https://api.telegram.org/botВАШ_ТОКЕН/setWebhook?url=https://ВАШ_WORKER_URL
```
