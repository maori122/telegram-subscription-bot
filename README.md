# 🤖 Telegram Bot для рекламных ссылок с отслеживанием подписок

Современный Telegram бот для рассылки рекламных ссылок с автоматическим отслеживанием подписок. Работает на **Cloudflare Workers** - бесплатно, быстро и надежно!

## ✨ Как это работает

### Для пользователей:

1. **Регистрация** - пользователь нажимает кнопку "✅ Зарегистрироваться"
2. **Получение ссылки** - админ рассылает рекламную ссылку всем
3. **Подписка** - пользователь подписывается по ссылке
4. **Скриншот** - отправляет скриншот подписки в бота
5. **Подтверждение** - получает автоматическое сообщение с количеством подписок

### Для администраторов:

1. **Авторизация** - `/admin пароль`
2. **Рассылка ссылок** - одна кнопка для отправки всем
3. **Управление подписками** - +1, -1 или обнулить для каждого пользователя
4. **Настройка ответа** - кастомный текст после получения скриншота
5. **Статистика** - просмотр всех пользователей и их подписок

## 🎯 Основные функции

### Для пользователей:
- ✅ Простая регистрация одной кнопкой
- 📸 Отправка скриншотов подписки
- 📊 Автоматический подсчет подписок
- 💬 Персонализированные ответы

### Для админов:
- 👥 **Список пользователей** - просмотр всех с количеством подписок
- 🔗 **Рассылка ссылок** - массовая отправка рекламных ссылок
- 💬 **Настройка ответа** - кастомный текст с переменной {count}
- ➕ **+1 подписка** - увеличить счетчик вручную
- ➖ **-1 подписка** - уменьшить счетчик
- 🔄 **Обнулить** - сбросить счетчик подписок
- 📸 **Уведомления** - получение алертов о новых скриншотах

## 🚀 Быстрый старт

### Автоматическая установка

**Windows:**
```bash
setup.bat
```

**Linux/Mac:**
```bash
chmod +x setup.sh
./setup.sh
```

### Ручная установка

```bash
# 1. Установка
npm install -g wrangler

# 2. Логин
wrangler login

# 3. KV namespace
wrangler kv:namespace create "USERS"
# Обновите ID в wrangler.toml

# 4. Секреты
wrangler secret put BOT_TOKEN
wrangler secret put ADMIN_PASSWORD

# 5. Деплой
wrangler deploy

# 6. Webhook
curl "https://api.telegram.org/botTOKEN/setWebhook?url=WORKER_URL"
```

## 📱 Использование

### Пользователь:

1. Отправить `/start`
2. Нажать "✅ Зарегистрироваться"
3. Получить ссылку от админа
4. Подписаться и отправить скриншот
5. Получить подтверждение

### Администратор:

1. Авторизация: `/admin ВАШ_ПАРОЛЬ` (установите свой пароль в Cloudflare Secrets)
2. Появится панель с кнопками:


## 💬 Настройка ответа

Админ может настроить текст, который получают пользователи после отправки скриншота:

**Пример:**
```
Зафиксировано {count} подписок! 
Спасибо за участие! 🎉
```

Переменная `{count}` автоматически заменяется на количество подписок.
