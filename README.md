# 🤖 Telegram Bot для рассылки

Современный Telegram бот с интерактивными кнопками для массовой рассылки сообщений. Работает на **Cloudflare Workers** - бесплатно, быстро и надежно!

## ✨ Особенности

- 🎯 **Интерактивные кнопки** - удобное управление через inline-клавиатуру
- 🔒 **Безопасная авторизация** - система паролей для админов
- 📢 **Массовая рассылка** - отправка сообщений всем пользователям одной кнопкой
- ✉️ **Персональные сообщения** - выбор пользователя из списка
- ☁️ **Cloudflare Workers** - бесплатный хостинг 24/7 без засыпания
- 💾 **Cloudflare KV** - надежное постоянное хранилище данных
- ⚡ **Мгновенный отклик** - edge computing по всему миру
- 🚀 **Автоматическая установка** - готовые скрипты для Windows/Linux/Mac

## 🚀 Быстрый старт

### Автоматическая установка (рекомендуется)

**Windows:**
```bash
setup.bat
```

**Linux/Mac:**
```bash
chmod +x setup.sh
./setup.sh
```

Скрипт автоматически:
- ✅ Установит Wrangler CLI
- ✅ Войдет в Cloudflare
- ✅ Создаст KV namespace
- ✅ Настроит секреты
- ✅ Задеплоит бота

### Ручная установка

```bash
# 1. Установка Wrangler
npm install -g wrangler

# 2. Логин в Cloudflare
wrangler login

# 3. Создание KV namespace
wrangler kv:namespace create "USERS"
# Скопируйте ID в wrangler.toml

# 4. Настройка секретов
wrangler secret put BOT_TOKEN
wrangler secret put ADMIN_PASSWORD

# 5. Деплой
wrangler deploy

# 6. Установка webhook
curl "https://api.telegram.org/botYOUR_TOKEN/setWebhook?url=YOUR_WORKER_URL"
```

## 📱 Использование

### Для пользователей:

1. Найдите бота в Telegram
2. Отправьте `/start` для регистрации
3. Используйте `/help` для справки

### Для администраторов:

1. Отправьте `/admin Yp~?L*W<a"v/(~mFm&[&`
2. Появятся кнопки управления
3. Нажимайте на кнопки для выполнения действий!

## 📋 Команды

| Команда | Описание |
|---------|----------|
| `/start` | Регистрация в боте |
| `/admin <пароль>` | Авторизация как администратор |
| `/help` | Справка по командам |

## ❓ Решение проблем

### Бот не отвечает:

```bash
# Проверьте логи
wrangler tail

# Проверьте webhook
curl "https://api.telegram.org/botTOKEN/getWebhookInfo"

# Проверьте секреты
wrangler secret list
```

### Ошибка KV:

1. Убедитесь, что KV namespace создан
2. Проверьте ID в `wrangler.toml`

Подробнее см. [CLOUDFLARE_DEPLOY.md](CLOUDFLARE_DEPLOY.md)
