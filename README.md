# Telegram Bot для рассылки

Бот для рассылки сообщений зарегистрированным пользователям с системой авторизации администраторов по паролю.

## 🚀 Быстрый старт (локально)

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Создайте `.env` файл:
```bash

```

3. Запустите бота:
```bash
python bot.py
```

## 📋 Команды бота

### Для пользователей:
- `/start` - Регистрация в боте
- `/admin <пароль>` - Авторизация как администратор
- `/help` - Помощь

### Для администраторов:
- `/users` - Список всех зарегистрированных пользователей
- `/broadcast <текст>` - Рассылка сообщения всем
- `/send @username <текст>` - Отправка конкретному пользователю
- `/send 123456789 <текст>` - Отправка по ID

## 🔒 Безопасность

- Пароли хешируются с bcrypt (12 раундов)
- Сообщения с паролями автоматически удаляются
- Авторизованные админы сохраняются в `admins.json`
