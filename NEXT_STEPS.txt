╔══════════════════════════════════════════════════════════════╗
║              ШАГ 1: ЗАГРУЗИТЕ НА GITHUB                      ║
╚══════════════════════════════════════════════════════════════╝

1. Создайте новый репозиторий на https://github.com/new

2. Выполните команды:

   git remote add origin https://github.com/ваш-username/название-репо.git
   git branch -M main
   git push -u origin main

╔══════════════════════════════════════════════════════════════╗
║         ШАГ 2: ДЕПЛОЙ НА CLOUDFLARE WORKERS                  ║
╚══════════════════════════════════════════════════════════════╝

ВАРИАНТ А: Автоматическая установка (РЕКОМЕНДУЕТСЯ)

   Windows:
   setup.bat

   Linux/Mac:
   chmod +x setup.sh
   ./setup.sh

ВАРИАНТ Б: Ручная установка

   1. npm install -g wrangler
   2. wrangler login
   3. wrangler kv:namespace create "USERS"
   4. Обновите ID в wrangler.toml
   5. wrangler secret put BOT_TOKEN
   6. wrangler secret put ADMIN_PASSWORD
   7. wrangler deploy

╔══════════════════════════════════════════════════════════════╗
║              ШАГ 3: УСТАНОВИТЕ WEBHOOK                       ║

Или откройте в браузере:
https://api.telegram.org/botВАШ_ТОКЕН/setWebhook?url=https://ВАШ_WORKER_URL
