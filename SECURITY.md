# 🔒 Безопасность

## ⚠️ ВАЖНО: Установка пароля

**НЕ ИСПОЛЬЗУЙТЕ пароли из примеров в документации!**

Все пароли в примерах - это только демонстрация. Вы ДОЛЖНЫ установить свой собственный надежный пароль.

## Как установить свой пароль

### Для Cloudflare Workers:

```bash
wrangler secret put ADMIN_PASSWORD
```

Когда система попросит, введите свой надежный пароль (минимум 16 символов).

### Генерация надежного пароля

Используйте один из этих способов:

**1. Онлайн генератор:**
- https://passwordsgenerator.net/
- Выберите: 20+ символов, буквы, цифры, спецсимволы

**2. Командная строка (Linux/Mac):**
```bash
openssl rand -base64 32
```

**3. PowerShell (Windows):**
```powershell
-join ((48..57) + (65..90) + (97..122) + (33..47) | Get-Random -Count 24 | % {[char]$_})
```

**4. Python:**
```python
import secrets
import string
alphabet = string.ascii_letters + string.digits + string.punctuation
password = ''.join(secrets.choice(alphabet) for i in range(24))
print(password)
```

## Требования к паролю

✅ Минимум 16 символов
✅ Буквы верхнего и нижнего регистра
✅ Цифры
✅ Специальные символы
❌ НЕ используйте словарные слова
❌ НЕ используйте личную информацию
❌ НЕ используйте пароли из примеров

## Хранение пароля

- ✅ Храните в менеджере паролей (1Password, Bitwarden, LastPass)
- ✅ Записывайте в надежном месте офлайн
- ❌ НЕ храните в коде
- ❌ НЕ коммитьте в Git
- ❌ НЕ публикуйте на GitHub

## Проверка безопасности

После установки проверьте:

```bash
# Убедитесь, что секрет установлен
wrangler secret list

# Должно показать:
# ADMIN_PASSWORD
# BOT_TOKEN
```

## Что делать если пароль утек

1. Немедленно смените пароль:
```bash
wrangler secret put ADMIN_PASSWORD
```

2. Проверьте логи на подозрительную активность:
```bash
wrangler tail
```

3. Если нужно, пересоздайте бота через @BotFather

## Дополнительные меры безопасности

### 1. Приватный репозиторий
Сделайте репозиторий на GitHub приватным, если храните там конфигурацию.

### 2. .gitignore
Убедитесь, что `.env` файлы не попадают в Git:
```
.env
.env.local
.env.production
```

### 3. Регулярная смена пароля
Меняйте пароль админа каждые 3-6 месяцев.

### 4. Мониторинг
Регулярно проверяйте логи:
```bash
wrangler tail
```

## Безопасность Cloudflare Secrets

Cloudflare Secrets:
- ✅ Зашифрованы
- ✅ Не видны в коде
- ✅ Не видны в логах
- ✅ Доступны только вашему Worker

## Контрольный список

Перед деплоем проверьте:

- [ ] Установлен уникальный пароль через `wrangler secret put`
- [ ] Пароль не содержится в коде
- [ ] Пароль не содержится в документации
- [ ] `.env` файлы в `.gitignore`
- [ ] Репозиторий приватный (или пароли удалены)
- [ ] Пароль сохранен в надежном месте

## Помощь

Если вы случайно опубликовали пароль:
1. Немедленно смените его
2. Проверьте историю Git
3. Если нужно, удалите историю коммитов с паролем

Для удаления из истории Git:
```bash
git filter-branch --force --index-filter \
  "git rm --cached --ignore-unmatch ФАЙЛ_С_ПАРОЛЕМ" \
  --prune-empty --tag-name-filter cat -- --all
```

**Но лучше просто сменить пароль!**
